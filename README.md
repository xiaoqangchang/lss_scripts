

# Loon/Surge/Stash 脚本插件集合

本仓库收集整理了适用于 Loon、Surge 和 Stash 的脚本插件，主要包含以下功能：

## 主要功能

- **广告拦截**：针对各类应用的广告屏蔽
- **VIP 解锁**：部分应用的会员功能解锁
- **去广告**：去除应用内的广告内容
- **功能增强**：提升应用使用体验

## 目录结构

- **Loon/**：适用于 Loon 的插件文件（.plugin）和脚本（.lsr）
- **Surge/**：适用于 Surge 的模块文件（.sgmodule）
- **Stash/**：适用于 Stash 的覆写文件（.stoverride）
- **js/**：通用 JavaScript 脚本文件
- **icon/**：应用图标资源

## 支持的应用

- **社交**：微信、微博等
- **音乐**：酷狗、网易云音乐、Spotify 等
- **视频**：芒果TV、南瓜影院等
- **工具**：百度地图、高德地图、12306 等
- **效率**：滴答清单、印象笔记等
- **流媒体**：Netflix、YouTube、Disney+、HBO 等

## 使用方法

### Loon
1. 打开 Loon 应用
2. 进入「配置」→「脚本」
3. 添加对应插件或脚本

### Surge
1. 打开 Surge 应用
2. 进入「配置」→「脚本」
3. 导入对应模块

### Stash
1. 打开 Stash 应用
2. 进入「覆写」页面
3. 添加对应覆写文件
